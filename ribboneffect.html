Ribbon Effect

The ::before and ::after pseudo-elements can be used for ribbons and other visual effects without images.

“ basic technique of adding visual effects with the ::before and ::after pseudo-elements. The content property is set to an empty string, creating an empty element that can be styled as a shape and then moved into position using absolute positioning. In this example, the shapes are 20px high, so they are accurately positioned below the heading by setting a negative bottom offset of the same amount. ”
From "Beginning CSS3" by David Powers
